# Building Babbel Speak — From Hackathon to Home Screen

A portfolio case study and AI product management course by **Franz Ardito**, Principal Product Manager at Babbel.

## What This Is

Babbel Speak is a conversational AI that helps beginner language learners overcome the fear of speaking. This course tells the story of how it was built — from a 2019 hackathon prototype to a 2025 home screen tab — and teaches the AI product management frameworks that made it possible.

**Two audiences, one document:**

- **Hiring managers** evaluating product thinking, technical fluency, and decision-making rigor
- **PMs** working through exercises to learn AI product management skills

## Files

| File | Purpose |
|------|---------|
| `SPEAK-PORTFOLIO.md` | Main document — 10 chapters (~11,000 words). Start here. |
| `tutor/system-prompt.md` | AI tutor setup — paste into any AI tool for interactive coaching |
| `exercises/` | Datasets and templates for hands-on exercises |

## The 10 Chapters

| # | Title | Lesson |
|---|-------|--------|
| 1 | The Beginner's Paradox | Problem Diagnosis — separate symptoms from root causes |
| 2 | 12 Hours That Changed Everything | Hackathon as Validation — cheap validation before big investment |
| 3 | Working with Water | Tech Derisking — capability maps and 3 categories of unknowns |
| 4 | 180 Conversations | Error Analysis — open coding, axial coding, saturation |
| 5 | The Fix Workflow | Small Batch Iteration — 5-step loop, same-day checks |
| 6 | Building the Scenario Evaluator | Manual to Automated — layered eval strategy |
| 7 | Calibrating the Judge | Eval Design — LLM-as-judge calibration, common failure modes |
| 8 | The Spike and the Truth | Quasi-Experiments — DiD, Prophet, spike-and-decay |
| 9 | The Ship Decision | Ship/Hold/Conditional — guardrails vs optimizations |
| 10 | What I'd Do Differently | Eval Culture — maturity ladder, ownership RACI |

## How to Use This Course

1. **Read `SPEAK-PORTFOLIO.md`** — the full narrative and frameworks
2. **Work through exercises** using the datasets in `exercises/`
3. **Optionally, load the tutor** — copy `tutor/system-prompt.md` into ChatGPT, Claude, or any AI tool for guided coaching

## No Dependencies

This course works with any AI tool. No platform-specific formats, no proprietary software. Just markdown, CSV, and you.
